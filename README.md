# Quick Translator

Translate selected text using Raycast AI and automatically replace it.

## Features

- **Auto-detect language**: Chinese â†” English translation
- **Quick replace**: Automatically replaces selected text with translation
- **Multiple languages**: Support English, Simplified Chinese, and Japanese

## Usage

1. Select any text in any application
2. Open Raycast and run "Translate Selected Text"
3. The selected text will be replaced with the translation

## Configuration

| Option | Description |
|--------|-------------|
| Auto (CN <-> EN) | Auto-detect: Chinese to English, English to Chinese |
| English | Always translate to English |
| Simplified Chinese | Always translate to Simplified Chinese |
| Japanese | Always translate to Japanese |

## Requirements

- Raycast Pro subscription (for AI features)
